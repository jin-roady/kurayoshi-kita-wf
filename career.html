進路サポート

夢を叶える3年間のロードマップ

入学直後からキャリア教育をスタート。
個別面談を重ね、生徒の希望に寄り添った指導を行います。

主な取り組み

  • 1年次：職業適性検査、進路ガイダンス
  • 2年次：大学・専門学校見学会、インターンシップ
  • 3年次：面接指導、小論文対策講座、共通テスト対策

進路実績

国公立大学

鳥取大学 / 島根大学 / 岡山大学 / 山口大学 / 公立鳥取環境大学 / 防衛大学校（理工学専攻） など

私立大学・短期大学

関西大学 / 近畿大学 / 京都産業大学 / 龍谷大学 / 岡山理科大学 / 鳥取短期大学 など

就職・公務員

公務員 自衛隊 / 鳥取県警 など
民間企業 日本郵便（中国支社） / JR西日本 / 小田急電鉄 / 花王コスメプロダクツ / ホテル玉泉 / トンボ倉吉工房 / エディオン など
